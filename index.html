Predicting Estrus Onset: Activity Profiles

Toggle the average 24-h activity profiles to explore sex differences.

1. What have you done so far?

I've set up a GitHub Pages-hosted prototype that displays my first interactive visualization: the mean 24-hour activity profiles of female and male mice under a 12 h light/dark cycle. I loaded the raw Excel sheets, cleaned and merged the minute-by-minute activity and temperature data in JavaScript, and manipulated it into a single JSON format for Chart.js. Users can toggle each curve on or off via checkboxes, giving immediate feedback on sex-specific rhythm differences. All HTML, CSS and JS files have been uploaded to the repository, and I've confirmed the page renders correctly across desktop and mobile.

Overall, this prototype establishes the core UI structure—header, controls, canvas, and write-up sections—and proves my data pipeline from raw sheets to chart. Next steps will embed my preprocessing outputs directly into the visualization and begin annotating known estrus deviations.

2. What will be the most challenging part to design and why?

The most challenging design aspect will be creating an engaging narrative that clearly shows why estrus onset matters and surprises viewers with insights from the data. Crafting a compelling hook using an “and-but-therefore” storytelling structure requires thoughtful pacing and transitions. Additionally, embedding the project video in a way that is visually prominent but avoids clutter will involve careful layout considerations.

Selecting visual encodings that are both expressive and intuitive—such as shading dark and light cycles or annotating estrus windows—will be critical. Ensuring interactive features like filtering options, detailed hover effects, and clear model confidence indicators are effective but not overwhelming is another significant hurdle.

Finally, refining the interactive elements to ensure they're bug-free, accessible (including keyboard navigation and responsive resizing), and cohesive with clear annotations and impactful takeaways will require meticulous attention to detail.
